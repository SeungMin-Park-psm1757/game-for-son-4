# Prompt Library

Use these as starting points for future image generation or handoff briefs. Keep outputs original and avoid asking for exact imitation of a named studio.

## Dock Icon Prompt

## Training Item Prompt

Create a polished 2D animation-style training item illustration for a pet-care mobile game, transparent background, one clear hero object, soft brush-like shading, subtle ambient glow, playful but readable silhouette, no pixel art, no text, no clutter.

## Sleep Choice Prompt

Create a cozy sleep-location illustration for a pet-care game UI, transparent background, warm hand-painted 2D style, soft nighttime lighting, calm safe atmosphere, limited props, strong silhouette, readable on a small mobile card.

## Stage Backdrop Prompt

Create a side-view pet stage backdrop for a mobile care game, hand-painted 2D animation feel, airy sky wash, soft layered ground, original whimsical nature mood, gentle lighting, open center space for pet character and speech bubble, no UI text, no clutter.
